# IMDb_Commands_Classification
Trying NLP project

## 数据清洗:
移除HTML标签：评论可能包含HTML标签，如<br>。
移除非字母字符：例如数字、标点符号等。
将所有文本转换为小写：这样可以确保词汇的统一性。

## 分词:
将评论分解成单词或标记。

## 去除停用词:
停用词是那些在文本中频繁出现但对分析没有太大意义的词，如“the”、“and”、“is”等。

## 词形还原:
将单词还原到其基本形式，例如将“running”还原为“run”。

## 向量化:
将文本转换为数字形式，常用的方法有TF-IDF、Word2Vec等。
